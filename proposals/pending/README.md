# Pending proposals

Use this directory for private MVP candidate trap records before they are promoted into `records/traps/`.

External contributors should usually start with a GitHub issue using the stuck-agent report template. Maintainers or agents can convert good reports into JSON candidates here.

Do not treat files in this directory as canonical Aigora guidance.

Before schema-shaping a markdown proposal into a candidate JSON record, include or identify: negative signals/false-positive boundaries, risk and human-gate notes, and at least one public or artifact source plan. These do not need to be perfect, but they reduce reviewer round-trips and accidental over-application.
